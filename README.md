# Game

Learning games and fun activities for kids with quizzes:

- Multiplication trainee

### Create the project:

npx create-expo-app my-app

### Run the project:

```
$ npm start
```

### Create Build:

1. eas login
2. eas build:configure // only for Android
3. eas build --platform android

### Run on the emulator:

Run the emulator
npx expo start -> s -> a

### Update SDK:

$ npm i expo@latest
$ npx expo i --fix
